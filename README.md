# Kaoako Vault — Mexxen AI 知識庫

> **打造合夥人等級的 AI Agent**

## 🎯 這是什麼

這是一個專為 **Mexxen** 打造的 AI 知識庫，使用 **Obsidian + Git** 架構管理。

**Kaoako** 是你的 AI 合夥人、二把手。它不是一個聽指令的工具，而是一個累積了你所有決策脈絡、能跟你討論、挑戰你想法、幫你做判斷的夥伴。
